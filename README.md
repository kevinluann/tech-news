**Portal de notícias** sobre tecnologia, desenvolvido com HTML5 e CSS3.

<hr />

</div>

## 🎯 Objetivo

Projeto de estudo para praticar:
- HTML5 semântico e estrutura de conteúdo
- CSS Grid Layout para estrutura principal
- CSS Flexbox para componentes específicos
- Organização modular de arquivos CSS
- Uso de variáveis CSS customizadas
- CSS nesting para melhor organização do código

## Estrutura da Página

- **Header** com navegação principal e categorias
- **Destaque** com notícia principal em destaque
- **Seções** de notícias por categoria

## Tecnologias
- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e layout com Grid e Flexbox
